# Représentation d'images en couleurs

## Description du projet

On va considérer quatre manières de représenter une image en couleur et donc quatre classes d'images :

- `BruteRasterImage` 
- `PaletteRasterImage` 
- `SparseRasterImage` 
- `VectorImage`
